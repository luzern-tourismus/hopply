# Hopply

### Installation

```
composer config repositories.hopply vcs https://github.com/luzern-tourismus/hopply
composer require luzern-tourismus/hopply
```

## Config

OPEN_AI_API_KEY

php -S localhost:9000 ./web

## Deployment

git pull
composer update
